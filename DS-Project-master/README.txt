There are two folders, of which the names are PureSoundex&GED and ModifyR.The PureSoundex&GED is for the first task. It implements the algorithm soundex and global edit distance. The main method is in Entrance.java. The second one, ModifyR, is for the task modifying r parameter, there are several *.java files in the folder. And the main method is in Entrance.java. For these two tasks, the results can be generated by just running the main method.
